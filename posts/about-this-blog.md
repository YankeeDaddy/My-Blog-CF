这是一个纯静态的个人博客，使用 HTML + CSS + JavaScript 构建。没有依赖任何框架，保持轻量和简洁。

### 为什么选择纯静态？

对于个人博客来说，静态页面有明显的优势：**快速**、**安全**、**易于部署**。不需要数据库，不需要服务器端逻辑，打开浏览器就能运行。

### 功能特性

- 📝 文章的创建、编辑和删除
- 💬 基于 Giscus 的评论系统（支持 GitHub 账号登录评论）
- 👍 文章点赞功能
- 🔍 全文搜索与分类过滤
- 🌗 暗色 / 明亮模式切换
- 📱 完整的移动端适配

> 最好的界面是没有界面——用户直接与内容互动，而不需要学习如何使用工具。
