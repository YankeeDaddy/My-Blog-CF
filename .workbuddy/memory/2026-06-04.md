### 2026-06-04 更新：点赞&评论数据跨设备持久化（Cloudflare Pages Functions）
- **问题根因**：`syncLikesToGithub()` 需要用户浏览器的 PAT Token，普通访客没有 → 点赞只存 localStorage → 换设备丢失
- **解决方案**：Cloudflare Pages Functions 做服务端代理
  - `functions/api/likes.js`：GET 读取 / POST 写入（SERVER-SIDE PAT）
  - `functions/api/discussions.js`：GET 获取评论统计（带认证，5000 req/hr）
- 前端全部改为优先走 `/api/*`，均有无感 fallback
- 用户需在 CF Pages Settings → Environment variables 中添加 `GITHUB_PAT`
- 推送 commit `aa3b610`

### 2026-06-04 更新：独立 Cloudflare Worker 方案（最终方案）
- **问题根因**：Cloudflare Pages Functions 在无构建静态站点中不工作，所有 `/api/*` 返回 404
- **解决方案**：创建独立 Cloudflare Worker (`peyblog-likes-api`)
  - `worker.js`：处理 `/api/likes` GET/POST 和 `/api/discussions` GET
  - `index.html`：添加 `API_BASE` 常量 = Worker URL，所有 API 调用指向 Worker
- **已清理**：删除 `_worker.js` 和 `functions/` 目录（不工作）
- **部署步骤**：见 `WORKER_DEPLOY.md`
